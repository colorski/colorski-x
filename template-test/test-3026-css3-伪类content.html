伪类content

兼容ie9+ ； 以下内容来自：css3 content 生成内容-w3cplus

content一般和:before,:after一起使用，用来生成内容(img和input没有该属性)，content的内容一般可以为以下四种：

none： 不生成任何值。
attr： 插入标签属性值
url： 使用指定的绝对或相对地址插入一个外部资源（图像，声频，视频或浏览器支持的其他任何资源）
string： 插入字符串

1. none

其实我们常用的clearfix就是应用了这个none了.

    .clearfix:after {
        content: "";
        visibility: hidden;
        display: block;
        font-size: 0;
        clear: both;
        height: 0;
    }
    * html .clearfix             { zoom: 1; } /* IE6 */
    *:first-child+html .clearfix { zoom: 1; } /* IE7 */
2. attr： 插入标签属性值

可以是普通文字

青，取之于蓝，而青于蓝；冰，水为之，而寒于水。

    .attr-title:after{
        content:attr(title);
        color:#f00;
    }
