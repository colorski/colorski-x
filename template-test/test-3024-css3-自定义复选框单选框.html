自定义复选框单选框

自定义的复选框-用的特殊字符形式的，也可用图片形式的，如单选框。

扩展： CSS3的content属性详解 css3 content 生成内容 特殊字符 

    input[type="checkbox"] + label {
        cursor: pointer;
    }
    .test1 input[type="checkbox"] {
        position: absolute;
        clip:rect(0,0,0,0);
    }
    .test1 input[type="checkbox"] + label::before {
        content: '\a0'; /*不换行空格*/
        display: inline-block;
        vertical-align:0;
        width: 14px;
        height: 14px;
        margin-right: 6px;
        border-radius: 2px;
        background: silver;
        text-align: center;
        line-height: 14px;
    }
    .test1 input[type="checkbox"]:checked + label::before {
        content: '\2714';
        background: green;
        color: #fff;
    }
    .test1 input[type="checkbox"]:focus + label::before {
        box-shadow: 0 0 .1em .1em #58a;
    }
    .test1 input[type="checkbox"]:disabled + label::before {
        background: #f2f2f2;
        box-shadow: none;
        color: #555;
    }
自定义的单选框-用图片做背景的形式
    /*单选框*/
    .test2 input[type="radio"] {
        position: absolute;
        clip:rect(0,0,0,0);
    }
    .test2 input[type="radio"] + label::before {
        content: '\a0'; /*不换行空格*/
        display: inline-block;
        vertical-align:0;
        width: 20px;
        height: 20px;
        margin-right: 6px;
        background: url(css/plugins/iCheck/red.png) -88px 0px no-repeat;
    }
    .test2 input[type="radio"]:checked + label::before {
        background-position:-110px 0px;
    }
    .test2 input[type="radio"]:disabled + label::before {
        background-position:-132px 0px;
    }
    .test2 input[type="radio"]:checked:disabled + label::before {
        background-position:-154px 0px;
    }
