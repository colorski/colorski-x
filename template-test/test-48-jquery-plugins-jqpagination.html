jqpagination插件

需要的文件：
jquery.jqpagination.min.js
jqpagination.css 样式比较少，可以整理到一个公用css文件里
用法：
html：
javascript: 
    $('#jqPager').pager({
		recordcount: data.count,
		pagesize: data.pageSize,
		pagenumber:data.page,
		buttonClickCallback:function(n) {
			getData(n);
		},
		firsttext: '首页',
		prevtext: '前一页',
		nexttext: '下一页',
		lasttext: '尾页',
		gotext:'跳转',
		recordtext: '共{0}页,{1}条记录'
	});

	以上代码一般放在getData()方法中的success中的插入之后；

	//需要的参数：
	count -- 数据总数；
	pageSize -- 每页展示数据数；
	page -- 当前页数；
