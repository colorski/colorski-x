	$(this).next('dl.cf').clone(true).appendTo('#paramBox');//这时如果被复制项填了值，也会复制这个值，可以这样：

$(selector).clone(true|false);
false - 默认。不复制事件处理程序；
true - 复制元素的同时复制事件处理程序；
如果选择学段后要获取相应的科目列表，即高中、初中、小学对应的科目是不一样的； 这时候要从学段下手，然后$(this).next('.subject')去获取当前学段下的科目；
