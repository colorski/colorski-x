自适应内部元素

需求：一个元素，内部有一张图片和一段图片的描述。图片尺寸不定，想让这个元素与图片宽度相适应，并居中显示。

code：

		
这是在楼顶的围观群众甲。这是在楼顶的围观群众甲。这是在楼顶的围观群众甲。这是在楼顶的围观群众甲。

		
		
这是在底部的吃瓜群众乙，当然我也可以是其它元素了。

一般效果是这样：

这是在楼顶的围观群众甲。这是在楼顶的围观群众甲。这是在楼顶的围观群众甲。这是在楼顶的围观群众甲。

这是在底部的吃瓜群众乙，当然我也可以是其它元素了。

分析：

众所周知，如果不给一个元素指定height，它会自适应其内容的高度。假如我们希望width也具有类似的行为，即上边的需求。

code：

		.result figure {  border: 1px dashed #ccc; padding: 15px;
			max-width: 300px;
			max-width: min-content;
			margin: auto;
		 }
		 .result figure > img { max-width: inherit; }

需求结果是这样：

这是在楼顶的围观群众甲。这是在楼顶的围观群众甲。这是在楼顶的围观群众甲。这是在楼顶的围观群众甲。

这是在底部的吃瓜群众乙，当然我也可以是其它元素了。

解释：

css3的css内部与外部尺寸模型为width和height属性定义了一些新的关键字。如min-content，这个关键字浏览器将解析为自适应这个容器内部最大的不可断行元素的宽度（即最宽的单词、图片或具有固定宽度的盒元素）

max-width:300px;是为适应旧浏览器的回退样式；

对应其它关键字：max-content fit-content
