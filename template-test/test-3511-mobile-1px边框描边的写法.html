细线边框的写法

一般的app边框描边的线都小于一像素，如果直接给1px的border，虽然是1px可是结果和app里的描边完全不一样“粗了”，以下是查找资料后总结的两个写法。

方法一：利用css3的linear-gradient

  • linear-gradient
  • linear-gradient
  • linear-gradient
	.listem1 li{  
		background-image: -webkit-linear-gradient(#eee 50%,transparent 50%);
		background-image: -moz-linear-gradient(#eee 50%,transparent 50%);
		background-image: -o-linear-gradient(#eee 50%,transparent 50%);
		background-image: linear-gradient(#eee 50%,transparent 50%);
		background-size:  100% 1px;
		background-repeat: no-repeat;
		background-position: bottom;
	}

方法二：利用CSS3阴影（box-shadow），就是用阴影做描边然后用伪类把多余的给遮罩着

  • box-shadow
  • box-shadow
  • box-shadow
	.listem2 li{
		box-shadow: inset 0 -1px 1px #ccc;
		background: #fff;
		margin-left: -1px;
		margin-bottom: 10px;
		position: relative;
	}
	.listem2 li::after{
		content:'';
		position: absolute;
		top:0;
		left: 0px;
		right: 0px;
		bottom: 0px;
		border: 1px solid #fff;
	}
