分割线写法

利用背景色负margin -- 推荐 -- 可自适应宽度
小小分隔线 巧用背景色实现
    	.line_01 {
    	    height: 1px;
    	    border-top: 1px solid #ddd;
    	    text-align: center;
    	}
    	.line_01 span{
    	    position: relative;
    	    top: -10px;
    	    background: #fff;
    	    padding: 0 20px;
    	}
利用border -- 如果固定宽度的可以用，并且文字多少也要固定
小小分隔线 单标签实现
    	.line_02 {
    	    padding: 0 20px 0;
    	    margin: 20px 0;
    	    line-height: 1px;
    	    border-left: 200px solid #ddd;
    	    border-right: 200px solid #ddd;
    	    text-align: center;
    	}
